# Authentication Background Image

## Required Image

The login and signup pages require a background image located at:
```
public/auth-background.jpg
```

## Specifications

- **Dimensions**: Recommended 1920x1080 or higher
- **Format**: JPG, PNG, or WebP
- **Subject**: Professional office, technology, or business-related imagery
- **Aspect Ratio**: 16:9 or similar landscape format

## Suggested Sources

You can use free stock images from:
- Unsplash (https://unsplash.com)
- Pexels (https://pexels.com)
- Pixabay (https://pixabay.com)

## Search Keywords

- "office technology"
- "business workspace"
- "modern office"
- "QR code scanning"
- "asset management"
- "warehouse inventory"

## Implementation

Simply add your chosen image as `auth-background.jpg` to the `public` folder. The image will automatically display on the left side of the login and signup pages (visible on tablets and larger screens).

If no image is provided, the gradient overlay will still display, but without a background image underneath.
